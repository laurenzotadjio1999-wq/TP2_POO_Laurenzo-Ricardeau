# Carnet d’Adresses – TP2_POO_Laurenzo_Ricardeau
# Description du projet

Ce projet est une application graphique (GUI) développée en Python avec PyQt6 et utilisant SQLite3 comme base de données.

•Elle permet de gérer un carnet d’adresses :

•Créer une table d’enregistrements,

•Ajouter, modifier, afficher et supprimer des contacts,

•Afficher les données dans un tableau interactif,

•Remplir automatiquement les champs du formulaire lors de la sélection d’une ligne.
